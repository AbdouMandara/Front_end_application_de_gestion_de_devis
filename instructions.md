# Application
C'est une application de dévis  dont le titre est "Gadgets-industries"

## Différentes pages
a - une page d'inscription / connexion
b - une page d'accueil
c - une page pour la section "électrique"
d - une page pour la section "solaire"
e - une page pour la section "électrique et solaire"
f - une page contact

### Description des différentes pages
a - page d'inscription / connexion :
- sur la page d'inscription : je veux qu'il y ait les champs : **nom**, **email**, **numero de tel**, **mot de passe**  et **confirmer le mot de passe**. en bas y a un <p> qui redirige vers la page de connexion  ce <p> contient "Avez-vous déjà un compte ? "
- sur la pâge de connxion : il y a les champs email et mot de passe

b - Page d'accueil : je veux qu'en dessous du <h1> il y ait 2 divs sur une ligne et en dessous une div qui est centré;  la 1ère div a pour titre : installation électrique , la 2ème div a pour titre : Installation solaire et la 3 e div qui est centré : installation électrique et solaire 
Quand je clique sur la div "installation électrique" je suis redirigé vers page_electrique.html
Quand je clique sur la div "installation solaire" je suis redirigé vers page_solaire.html
Quand je clique sur la div "installation électrique et solaire" je suis redirigé vers page_solaire_et_electrique.html

c- sur page_electrique.html : je veux qu'il y ait un formulaire avec les champs : nom, numero de tel, produit qui est un select dont les options sont :  dans unn fichier excel ainsi que la quantité du produit il y a un bouton pour ajouter un autre produit, quant je clique sur ce bouton j'ai un nouveau champ pour ajouter un autre produit, et un bouton pour soumettre le formulaire
d - page_solaire.html je veux qu'il y ait un formulaire avec les champs : nom, numero de tel, produit qui est un select dont les options sont :  dans un fichier excel ainsi que la quantité du produit , l'amperage qui est de type number et un input où on affiche quantité * amperage (ce input n'est pas modifiable) il y a un bouton pour ajouter un autre produit, quant je clique sur ce bouton j'ai un nouveau champ pour ajouter un autre produit, et un bouton pour soumettre le formulaire
e - une page pour la section "électrique et solaire" tu fias une intersection entre les deux formulaires
